1. 생성자(constructor): 객체가 생성될 때 객체 초기화 메소드
	- 클래스 이름과 동일
	- 반환 값 없음
	- 필드에 초기값 부여 or 특별한 초기화 절차

2. 기본 생성자(default constructor): 매개변수가 없는 생성자
	- 생성자를 (하나도) 정의하지 않으면 자동으로
	  기본 생성자를 만들어줌
	- 자동적으로 모든 멤버 변수 초기화
	   int = 0,  참조변수 = null,  boolean = false

3. this 참조 변수: 현재 객체 자신을 가리키는 참조 변수
	- 컴파일러에서 자동으로 생성
	- 생성자의 매개변수 이름과 필드 이름 구별

4. this(): 다른 생성자를 의미
	- 생성자 코드에서만 사용 (일반 메소드 불가)
	- 동일한 클래스에서 다른 생성자를 호출할 때 사용
	- 반드시 생성자의 첫 문장에서 사용
